# QudzMotors

QudzMotors adalah aplikasi web yang dirancang untuk mengelola informasi dan layanan terkait kendaraan bermotor. Proyek ini bertujuan untuk menyediakan platform yang efisien dan user-friendly bagi pengguna untuk mengakses berbagai informasi dan layanan otomotif.

## Fitur

- **Halaman Utama**: Menampilkan informasi umum dan layanan utama.
- **Tentang Kami**: Informasi mengenai latar belakang dan tujuan QudzMotors.
- **Kontak**: Formulir untuk menghubungi tim QudzMotors.
- **Berita**: Update dan berita terbaru seputar dunia otomotif.
- **Admin**: Panel administrasi untuk mengelola konten dan data.

## Teknologi yang Digunakan

- **Frontend**: HTML, CSS, JavaScript
- **Database**: [Firestore dari Firebase](https://firebase.google.com/products/firestore)

> **Catatan**: Bagian database pada proyek ini menggunakan Firestore dari Firebase untuk menyimpan dan mengelola data secara real-time dan skalabel.
